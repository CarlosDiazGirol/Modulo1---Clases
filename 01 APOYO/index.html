Tarea 7: Crea un Formulario de Registro

Objetivo: Practicar formularios con asociación correcta label-input

Instrucciones:

Crea un formulario que contenga:

Tarea 10: Documento HTML Completo - Desafío Final

Objetivo: Crear un documento HTML completo y bien estructurado desde cero

Instrucciones:

Crea un archivo HTML nuevo llamado mi-pagina-completa.html que incluya:

En el HEAD:

  • <html lang="es">
  • Meta charset UTF-8
  • Meta viewport
  • Meta description con una descripción de la página
  • Title descriptivo
  • Favicon (usa un placeholder si no tienes uno)

En el BODY:

  • Header: Con título h1 y navegación con 4 enlaces
  • Main: Con al menos 3 sections que incluyan:
    • Section 1: Sobre ti (con h2, párrafo e imagen)
    • Section 2: Habilidades (con h2 y lista desordenada de 5 items)
    • Section 3: Proyectos (con h2 y 2 articles, cada uno con h3, párrafo e imagen)
  • Footer: Con copyright, enlaces de redes sociales (inventados) y año actual

Requisitos de calidad:

  • Todas las etiquetas deben estar correctamente cerradas
  • Código indentado (sangrado) correctamente
  • Todas las imágenes con atributo alt
  • Solo un h1 en toda la página
  • Jerarquía de encabezados respetada (h1 > h2 > h3)
  • Usar etiquetas semánticas (no usar div para todo)

Desafío extra (opcional):

Añade un formulario de contacto en una section adicional con nombre, email y mensaje.

Crea tu documento en un archivo separado

Esta tarea debe hacerse en un nuevo archivo HTML, no en este documento.

Resumen: Lo que Has Practicado

  1. Encabezados: h1, h2, h3, h4, h5, h6
  2. Formatos de texto: strong, em, mark, del, ins, code
  3. Listas: ol (ordenadas) y ul (desordenadas)
  4. Enlaces: internos, externos, con target
  5. Imágenes: src, alt, width, height
  6. Semántica: header, nav, main, section, article, footer
  7. Formularios: input types, label-for, radio, checkbox
  8. Formatos avanzados: br, hr, small, blockquote, pre, sub, sup
  9. Documento completo: Estructura HEAD y BODY correctas
