# Introducción a HTML
## Objetivos de Aprendizaje

- Entender qué es HTML y su rol en el desarrollo web
- Conocer la estructura básica de un documento HTML
- Comprender la importancia de las etiquetas semánticas
- Aprender a insertar imágenes con atributos accesibles
- Crear formularios con asociación correcta entre labels e inputs
- Uso de herramientas externas de almacenamiento de imagenes (cloudinary)
- Uso de IAs como copiloto de aprendizaje (chatGPT/Gemini)

## Archivos del Proyecto

- `index.html` - Página principal con el ejercicio

### Tarea 1: Crea los 6 Encabezados
Practicar la jerarquía h1 hasta h6 con textos específicos

### Tarea 2: Crea un Párrafo con Diferentes Formatos
Escribir un párrafo usando: strong, em, mark, del, ins, code

### Tarea 3: Crea Listas Ordenadas y Desordenadas
- Lista ordenada (ol) con 5 pasos para aprender HTML
- Lista desordenada (ul) con 4 tecnologías web

### Tarea 4: Crea Enlaces Internos y Externos
- Enlaces externos con y sin target="_blank"
- Enlaces internos con anclas (#)
- Enlaces a otras páginas del sitio

### Tarea 5: Inserta Imágenes con Atributos Correctos
Insertar 2 imágenes con src, alt, width y height

### Tarea 6: Crea una Estructura con Etiquetas Semánticas
Construir un blog usando: header, nav, main, section, article, footer

### Tarea 7: Crea un Formulario de Registro
Formulario completo con:
- Inputs: text, email, password, date
- Checkboxes y radio buttons
- Textarea y botón submit
- Asociación correcta label-input (for/id)

### Tarea 8: Usa Formatos de Texto Avanzados
Practicar: br, hr, small, blockquote, pre, sub, sup

### Tarea 9: Documento HTML Completo - Desafío Final
Crear desde cero un archivo HTML completo con HEAD correcto y estructura semántica en BODY

---

**Cada tarea tiene:**
- Objetivo claro
- Instrucciones específicas paso a paso
- Ejemplos cuando son necesarios
- Área para escribir tu solución
