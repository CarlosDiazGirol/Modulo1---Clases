## Estructura de los Ejercicios

### EJERCICIOS CSS

#### Ejercicio 1: Formas Basicas
**Objetivo:** Crear elementos con diferentes formas

**Conceptos:**
- width y height
- background-color
- border-radius para circulos (50%)
- display: flex para organizar elementos


#### Ejercicio 2: Centrado con Flexbox
**Objetivo:** Centrar elementos horizontal y verticalmente

**Conceptos:**
- display: flex
- justify-content: center (horizontal)
- align-items: center (vertical)
- height del contenedor

#### Ejercicio 3: Bordes y Sombras
**Objetivo:** Agregar bordes y efectos de sombra

**Conceptos:**
- border: 3px solid color
- border-radius para esquinas redondeadas
- box-shadow para profundidad

#### Ejercicio 4: Flexbox Avanzado
**Objetivo:** Organizar elementos con propiedades avanzadas de Flexbox

**Conceptos:**
- flex-wrap para permitir saltos de linea
- justify-content: space-between para distribuir espacio
- gap para espaciado entre elementos
- Crear layouts responsivos sin Grid

#### Ejercicio 5: Transiciones y Hover
**Objetivo:** Crear efectos interactivos

**Conceptos:**
- transition para suavizar cambios
- :hover para efectos al pasar cursor
- transform: scale() y rotate()
- cursor: pointer

---

### EJERCICIOS JAVASCRIPT

#### Ejercicio 6: Recorrer Arrays con For
**Objetivo:** Aprender el bucle for basico

**Conceptos:**
- Declarar arrays
- Propiedad .length
- Bucle for (inicializacion, condicion, incremento)
- Acceder a elementos con [indice]

#### Ejercicio 7: Bucles con Condicionales
**Objetivo:** Combinar for con if/else

**Conceptos:**
- Operador modulo (%)
- Condicional if/else
- Contadores dentro de bucles
- Clasificar elementos

#### Ejercicio 8: Operadores Matematicos
**Objetivo:** Realizar operaciones matematicas

**Conceptos:**
- Suma (+), resta (-), multiplicacion (*), division (/)
- Acumuladores
- Operadores de comparacion (>, <, >=, <=)
- Encontrar maximos y minimos

#### Ejercicio 9: Recorrer Strings
**Objetivo:** Trabajar con strings como arrays de caracteres

**Conceptos:**
- Propiedad .length de strings
- Acceder a caracteres con [indice]
- Comparar caracteres
- Bucles anidados

#### Ejercicio 10: Ejercicio Completo
**Objetivo:** Combinar todos los conceptos aprendidos

**Conceptos integrados:**
- Arrays de numeros
- Bucles for
- Condicionales multiples (if/else if/else)
- Operadores matematicos
- Contadores y acumuladores
- Comparaciones
