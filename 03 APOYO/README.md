# Ejercicios de Apoyo: Arrays, Objetos y Métodos

Este conjunto de ejercicios está diseñado para practicar intensivamente iteraciones, bucles, condicionales y métodos de arrays. Todo se ejecuta en consola para enfocarse 100% en la lógica de JavaScript.

## Contenido de los Ejercicios

**Ejercicios 1-10: Todos en consola**
1. For + Condicionales
2. For con Arrays de Objetos
3. forEach
4. map
5. filter
6. Combinar filter + map
7. find, some, every
8. reduce
9. Arrow Functions
10. Ejercicio Integrador Completo
