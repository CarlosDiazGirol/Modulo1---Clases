# Introducción a HTML
## Contenido de la Clase

## Objetivos de Aprendizaje

- Entender qué es HTML y su rol en el desarrollo web
- Conocer la estructura básica de un documento HTML
- Comprender la importancia de las etiquetas semánticas
- Aprender a insertar imágenes con atributos accesibles
- Crear formularios con asociación correcta entre labels e inputs
- Uso de herramientas externas de almacenamiento de imagenes (cludinary)
- Uso de IAs como copiloto de aprendizaje (chatGPT/Gemini)

## Archivos del Proyecto

- `index.html` - Página principal con estructura semántica
- `contacto.html` - Página con formulario de contacto
