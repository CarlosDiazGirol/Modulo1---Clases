# HTML, CSS y JavaScript: Creando un Portafolio Interactivo

## contenido de la clase
## Objetivos de Aprendizaje

### HTML
- Estructura semantica completa (header, main, footer)
- Uso de navegacion y enlaces internos
- Preparacion de contenedores para contenido dinamico

### CSS
- Implementar reseteo CSS correcto
- Comprender el modelo de caja (box model)
- Dominar Flexbox para layouts modernos
- Importar y usar Google Fonts
- Position fixed para header persistente
- Responsive design con media queries
- Crear menu hamburguesa funcional
- Mantener footer siempre abajo con flex

### JavaScript
- Trabajar con arrays
- Crear y usar funciones
- Bucles
- Condicionales para logica
- Manejo de strings y numbers

## Archivos del Proyecto

- `index.html` - Estructura HTML del portafolio con 4 cards estaticas
- `css/reset.css` - Reseteo CSS en archivo separado
- `css/styles.css` - Todos los estilos CSS organizados (con @import de Google Fonts)
- `js/script.js` - Ejemplos de JavaScript basico (arrays, bucles, condicionales, funciones)
